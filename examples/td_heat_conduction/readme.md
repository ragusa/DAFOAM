Under the subfolder in this directory, the python driver for each set of OF cases is hold.
The contents in each subfolders is as follows:
* [one-region-uniform-DT](./one-region-uniform-DT) contains the driver for time-dependent data-assimilation applied to heat conduction cases where the heat plate has a single DT in the entire geometry

* [three-region-heterogenous-DT](./three-region-heterogenous-DT) contains the driver for time-dependent data-assimilation applied to heat conduction cases where the heat plate has three different regions with three different DTs
